# 🎨 Дизайнер

UI/UX дизайнер AI Pilot — субагент для работы над интерфейсом веб-чата.

## Как вызвать

Скажи **Зеро**: `"попроси дизайнера сделать X"`

Зеро спавнит дизайнера с полным контекстом и релевантными скиллами.

## Что умеет

- Редизайн компонентов чата
- Стилизация, темизация, анимация
- Mobile/responsive вёрстка
- Работа с CSS-переменными
- Аудит и полировка UI
- Пуш изменений на GitHub

## Контекст

- `SOUL.md` — характер и тон
- `MEMORY.md` — память о проекте
- `TOOLS.md` — инструменты, скиллы, команды
